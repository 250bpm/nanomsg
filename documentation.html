Other languages

If native binding for your language doesn't exist you may try to use ZeroMQ binding instead. Check the list of ZeroMQ language bindings here:

http://www.zeromq.org/bindings:_start

For instructions on using nanomsg with ZeroMQ language bindings check this manual page:

http://nanomsg.org/v0.0.0/nn_zmq.7.html
